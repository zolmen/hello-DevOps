# hello-DevOps

Egyszerű .NET 8 webalkalmazás, amely HTTP-n egy szöveget ad vissza.

## Előfeltételek

- .NET 8 SDK
- Git

## Build

1. Klónozd a repót:
   `git clone https://github.com/zolmen/hello-DevOps.git`
2. Lépj be a mappába:
   `cd hello-DevOps`
3. Navigálj a projekt mappájába:
   `cd src/HelloDevOps`
4. Futtasd a buildet:
   `dotnet build`

## Lokális futtatás

1. A projekt mappájában:
   `cd src/HelloDevOps` (ha még nem ott vagy)
2. Futtasd:
   `dotnet run`
3. Böngészőben nyisd meg:
   `http://localhost:8080`
